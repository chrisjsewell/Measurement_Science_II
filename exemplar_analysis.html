Setting Up Python¶

The recommended manner to analyse scientific data is using the Anaconda Scientific Python Distribution, available for free download. When using this it is also suggested that you setup an isolated Python environment so that you can control what version of third-party packages you are using. This is acheived in the command prompt as follows;

conda create --name mm2 ipython tornado pyzmq numpy matplotlib pandas

Next you activate your environment and open this IPython Notebook, with an option to render data plots within the notebook.

source activate mm2
cd path/to/notebook

def read_conc_intens_data(data_files, conc_conversion=1.0):
    """ conc_conversion is conversion required to for Molar units"""
    data_concs = []
    data_intens = []
    for f in data_files:
        d = np.loadtxt(f, skiprows=2)
        x, y = d[:,0]*conc_conversion, d[:,1]
        if data_concs == []:
            data_concs = x
        else:
            # check all files have same concentration set
            if not (data_concs == x).all():
                raise IOError('{0} has a different concentration set'.format(f))
        data_intens.append(y)
    return data_concs, np.array(data_intens)
